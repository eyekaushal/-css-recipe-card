# Dhaba Menu CSS Project

A simple beginner-friendly HTML and CSS project that displays Indian dessert recipes in a clean and visually appealing layout.

This project is designed to practice basic CSS concepts like layout design, spacing, shadows, and typography.

Features

- Simple responsive layout
- Recipe cards with images
- Ingredients and step-by-step recipe section
- Clean UI inspired by Indian dhaba menus
- Box shadows and card styling using CSS

Technologies Used

- HTML5
- CSS3
